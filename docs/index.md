# Welcome to prodsys

This is the documentation for the prodsys package. A package for modeling, simulating, and optimizing production systems.

## Installation

To install the package, run the following command in the terminal:

```bash
pip install prodsys
```

Please note that the package is only compatible with Python 3.11 or higher.

## Getting started

The package is designed to be easy to use. The following example shows how to create a simple production system and simulate it.

```python
import prodsys.express as psx



```

For more elaborate guides that cover more of the package's functionality, please see the [tutorials](Tutorials/tutorial_0_overview.md). For a complete overview of the package's functionality, please see the [API reference](API_reference/API_reference_0_overview.md).
